
2. 用任何瀏覽器打開 `index.html`：

## 📥 導出 PDF

### 方法 1：使用瀏覽器
1. 開啟網站：https://MinshiXiao.github.io/ai-projects/
2. 按 `Cmd/Ctrl + P` 打開列印對話框
3. 選擇 "另存為 PDF"
4. 設定：
   - 邊距：標準
   - 背景圖形：✓ 勾選
   - 頁首/頁尾：取消勾選
